Код на Python для вычисления разных уравнений, примеров, построении графиков и т.д.

При запуске не забудьте поменять директорию файла в start.bat!

При запуске кода у вас появится отдельное окно, в котором будет справочник для ввода символов, решение и ответ, построение графиков. Важно помнить, что чем сложнее процесс вычисления, тем дольше может загружаться ответ.
